🛒 Online Shopping System
A simple Java-based Online Shopping Simulation

📜 Description
This project is a basic Online Shopping System implemented in Java.
It allows users to add products to their cart and simulates purchases using object-oriented principles.

📂 Features
✅ Customer & Product Management
✅ Shopping Cart System
✅ Credit Balance Tracking
✅ Uses the Facade Design Pattern for easy maintenance

🛠️ Technologies Used
Java 21
Object-Oriented Programming (OOP)
Git & GitHub for Version Control
